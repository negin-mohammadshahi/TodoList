# To-Do List App 📝

A simple command-line To-Do List application built with C#.

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Simple and clean interface

## Technologies
- C#
- .NET

## How to Run
1. Open the project in Visual Studio
2. Press Ctrl + F5 to run
